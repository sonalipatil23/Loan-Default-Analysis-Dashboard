# Loan Default Analysis Dashboard

## Project Overview
This project analyzes loan default patterns, applicant demographics, and financial risk using Power BI.

## Tools Used
- Power BI
- SQL
- Microsoft Excel

## Dashboard Pages
1. Loan Default Overview
2. Applicants Demographic & Financial Profile
3. Financial Risk Metrics

## Key Insights
- Loan amount by purpose
- Default rate by employment type
- Average income analysis
- Credit score analysis
- Year-over-Year loan default trends
